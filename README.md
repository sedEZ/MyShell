# Myshell
A simple shell program that support command "ls" , "echo" , "sleep" and some basic commands.

Also , type "quit","exit","logout","bye" can be used to exit the process;

Useage:

    {$pwd}/shell [commannds] [&]
    
## Rewrote command list:

    -ls
    -echo
    -sleep
    -exit
    -cd
    -pwd
    -hostname
